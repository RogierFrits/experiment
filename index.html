Experiment: Repeating Computational Research

Target Audience

Researchers and students involved with computational research. Some command-line experience is recommended, but not required.

Why join?

By joining this experiment you contribute to an ongoing Master Thesis project carried out at the Vrije Universiteit. If you have any questions or request more information, please use the e-mail below.

How do I join?

This experiment is about repeating an existing research. Please read the following scenario carefully and press START.

Technical Requirements

  • Computer with Linux or MacOS operating system
  • Computer with super user (admin) rights

As you were reading in an old journal of Neurophysiology from 2013 you have a brilliant new insight on the model for ‘Interaction between cognitive and motor cortico-basal ganglia loops during decision making’ by M. Guthrie, A Leblois, A. Garenne, T. Boraud.

Luckily for you, the code for this model has recently been uploaded to GitHub. You found a splendid tutorial and decide to stick to that approach. You first want to successfully run the existing code before making the changes of your brilliant new insight.

With the help of Google, you estimate it will take you roughly 15 minutes. After that you will report your findings by answering a few questions. If you are really stuck and want to give up please remember the step you’ve reached and time it took you. Then continue answering the questions as well.

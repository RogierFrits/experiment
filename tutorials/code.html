Tutorial

  1. Download or clone https://github.com/RogierFrits/ReScience-Entry-Topalidou-Rougier-2015 to your local computer.
  2. Read the README.md of the code directory: link
  3. Download and install python if you donâ€™t have it already.
  4. Download pip.
    (Tip: Using Google is allowed. With MacOS you can use 'easy_install', with Linux 'apt-get'.)
  5. Download the project requirements: numpy, cython and matplotlib as described in the README.md file.
  6. Open the code directory in a terminal and compile the project with:
    python setup.py build_ext --inplace
    This will print some warnings and recommendations even with all the dependencies installed. You can ignore these.
  7. Delete figure-1.pdf from the code directory.
    • Open terminal
    • Go to code directory of the project
    • Generate figure-1.pdf with the command:
      python single-trial.py

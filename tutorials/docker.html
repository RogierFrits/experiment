Tutorial

  1. Download or clone https://github.com/RogierFrits/ReScience-Entry-Topalidou-Rougier-2015-2 to your local computer.
  2. Read the README.md of the code and docker directory.
  3. Download and install Docker if you don’t have it already. (Link is in the README of the docker directory)
  4. First we are going to create the base Docker container. Open a terminal and go to the docker/base directory of the project. Apply the following command:
    docker build –t topalidou-rougier-2015-base . (The dot refers to the Dockerfile)
  5. Type docker images to see the images. This should now list ubuntu and topalidou-rougier-2015-base.
  6. Now we are going to build the release container. Open a terminal and go to the root of the project. Apply the command:
    docker build –t topalidou-rougier-2015 . (mind the dot)
  7. docker images should now also list topalidou-rougier-2015.
  8. Let’s run the container:
    docker run –it topalidou-rougier-2015
  9. Type ls to see the files and folders.
  10. Remove figure-1.pdf with rm figure-1.pdf and confirm with ls.
  11. Generate figure-1.pdf with:
    python single-trial.py and confirm with ls.

Congratulations! Please answer the questions now.

Tips: you can exit the container by typing exit. You can remove the images with the docker rmi command.
